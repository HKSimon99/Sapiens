# 📓 세션 로그

> **존재 이유**: AI는 세션마다 기억이 끊긴다.
> 이 파일이 연속성의 다리. 세션 시작 시 마지막 항목 읽으면 어디까지 했는지 안다.

---

## 📐 사용 흐름

```
세션 시작
  ↓
이 파일 마지막 항목 읽기 → "다음 세션 첫머리에" 확인
  ↓
분산 복습 캘린더 → 오늘 D+? 도달한 것 회상 테스트
  ↓
새 학습 진행 (도메인 + 개념)
  ↓
세션 끝 → 항목 추가 + 모든 관련 파일 업데이트
```

---
