# 📓 세션 로그

> **존재 이유**: AI는 세션마다 기억이 끊긴다.
> 이 파일이 연속성의 다리. 세션 시작 시 마지막 항목 읽으면 어디까지 했는지 안다.

---

## 📐 사용 흐름

```
세션 시작
  ↓
이 파일 마지막 항목 읽기 → "다음 세션 첫머리에" 확인
  ↓
분산 복습 캘린더 → 오늘 D+? 도달한 것 회상 테스트
  ↓
새 학습 진행 (도메인 + 개념)
  ↓
세션 끝 → 항목 추가 + 모든 관련 파일 업데이트
```

---

## 작성 형식

```
### YYYY-MM-DD (요일) — Phase X / Day N
- **시간**: __분
- **AI 협업 모드**: Centaur / Cyborg / Solo
- **다룬 도메인·개념**:
- **핵심 한 줄**:
- **회상 점수**: D+? 복습 — ⭐⭐⭐⭐⭐

**업데이트한 파일**:
- ☐ 개념노트 (atomic note 추가)
- ☐ AI 협업 노트 (패턴 발견)
- ☐ 배경지식 라이브러리 (항목 추가)
- ☐ 나의 논증 기록 (글 작성)
- ☐ 나의 관점 (명제 추가/갱신)
- ☐ 읽기 라이브러리 (책·진도)

**새 발견·연결**:

**막힌 지점**:

**다음 세션 첫머리에**:

**분산 복습 등록**:
- D+1 (___): YYYY-MM-DD
- D+3 (___): YYYY-MM-DD
- D+7 (___): YYYY-MM-DD
- D+14 (___): YYYY-MM-DD
- D+30 (___): YYYY-MM-DD
- D+90 (___): YYYY-MM-DD
```
