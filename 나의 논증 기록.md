# ✍️ 나의 논증 기록

> **Paul Graham**: 글쓰기는 생각의 복사본이 아니다. 쓰는 과정에서 새로 발견하는 것.
> **좋은 글의 기준** — 내가 먼저 놀랐는가?
> 단순한 언어 = 강한 사고. 어려운 단어 = 약한 사고를 숨기는 갑옷.
