# 📖 읽기 라이브러리

> 다시 시작하는 독서를 헛되이 하지 않기 위해.
> **빠르게 많이 읽기 < 천천히 깊이 읽고 하나라도 내 것으로.**

---

## 📐 읽기 원칙

1. **밑줄보다 회상** — 읽고 덮은 뒤 안 보고 핵심 말하기
2. **하나의 깨달음 > 백 개의 형광펜**
3. **반대 의견과 함께** — 이 저자의 가장 강한 반박은?
4. **즉시 적용** — 읽은 날 안에 한 번 써먹기
5. **D+21 재회상** — 3주 후 다시 덮고 핵심 말하기

## 🔄 읽기 → 사유 → 기록 순환

```
읽는다
  ↓
책 덮고 회상 ← 이게 진짜 학습
  ↓
중요하면 → 「개념노트.md」로 atomic note 작성
  ↓
가벼우면 → 「배경지식 라이브러리.md」 항목 추가
  ↓
생각이 바뀌었나? → 「나의 관점.md」 갱신
  ↓
글로 써보기 → 「나의 논증 기록.md」
  ↓
D+21 재회상 → 진짜 남았는지 검증
```
